Machine learning is a subset of artificial intelligence that allows systems to learn and improve from experience without being explicitly programmed.

Deep learning is a subfield of machine learning that uses neural networks with many layers to model complex patterns in data.
Supervised learning is a type of machine learning where the model is trained on labeled input-output pairs.
Unsupervised learning is a type of machine learning that finds hidden patterns in data without predefined labels.
Reinforcement learning is a type of machine learning where an agent learns by interacting with an environment and receiving rewards or penalties.
